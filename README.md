É uma calculadora web moderna, desenvolvida com foco em proporcionar uma experiência visual agradável e interações suaves.

O projeto explora design de interface, microinterações e animações, trazendo uma calculadora com estética inspirada no estilo **kawaii**, utilizando tons suaves de roxo e rosa e transições fluidas.

Mais do que apenas realizar cálculos, o objetivo do projeto é praticar **arquitetura de componentes, gerenciamento de estado e criação de experiências de usuário no desenvolvimento front-end.**

#### ◌ Funcionalidades

✔ Números de **0 a 9**  
✔ **Decimal (.)**  
✔ Operações básicas

- Adição (+)
- Subtração (-)
- Multiplicação (×)
- Divisão (÷)

✔ **Porcentagem (%)**  
✔ **Raiz quadrada (√)**  
✔ Botão de limpeza **C**  
✔ Resultado com **=**

Além disso:

animações suaves com **Framer Motion**  
interface estilizada com **Tailwind CSS**  
layout responsivo para diferentes tamanhos de tela
